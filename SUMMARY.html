End-of-Turn (EOT) Detection Speedrun

Final Performance Progression:
Run 1 (Silence Timer Baseline): 1600 ms delay
Run 2 (Absolute Prosody + Linear): 1190 ms delay
Run 3 (Relative Prosody + Linear): 1322 ms delay
Run 4 (Relative Prosody + Non-Linear Tree): 235 ms delay
Run 5 (Librosa Spectral Features + Non-Linear Tree): 100 ms delay

1. Solution Architecture & Scientific Method

As documented in my RUNLOG.md, I progressed through 5 isolated experiments. I first proved that absolute volume is a poor predictor compared to relative volume drops (Run 2 vs 3). I then proved that linear models fail on acoustic feature sets, requiring a switch to a Random Forest (Run 4).

The final architecture (Run 5) expands beyond simple volume tracking. It leverages librosa to extract a 26-dimensional feature vector, focusing heavily on Zero Crossing Rate (ZCR) and Spectral Centroid. When a human finishes a thought, their voice naturally loses high-frequency brightness (dropping Centroid) and transitions into breathy, unvoiced sounds (spiking ZCR). This timbral shift generalizes across languages better than basic English intonation rules.
